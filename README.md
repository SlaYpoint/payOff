# payOff

## Quick setup

1. Clone this repo
```
git clone https://github.com/SlaYpoint/payOff.git && cd payOff
``` 
2. Install required depedencies.

```
npm run build
```
3. Rename the `env.sample` to `env` and add

```
REACT_APP_API_TOKEN=yourtoken
```
4. To start development server

```
npm run start
```
